======

```
1. parameter 1 : alas
2. parameter 2 : tinggi
```

- Print
```
a. Print Jumlah parameter yang dimasukan
b. Print alas
c. Print tinggi
d. Print Keseluruhan total parameter
e. Print PID program
```
